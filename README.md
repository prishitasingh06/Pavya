# Pavya
Personal wellness and menstrual cycle tracking app with period prediction, ovulation tracking, and fertile window forecasting 

Pavya is a simple, touch-friendly period tracker that helps you monitor your menstrual cycles, predict ovulation, and visualize fertile windows with a modern iOS-style interface. Track multiple cycles securely on your device and see your cycle at a glance using interactive cards and a mini calendar.

## Meaning of Pavya

Pavya is derived from Sanskrit, meaning “pure" or “wholesome.” It conveys a sense of clarity, wellness, and natural balance. In the context of my app, it symbolizes purity of health, mindfulness, and the natural rhythm of the body.

##  Features
- User Authentication
- Login/logout functionality using localStorage
- Cycle Tracking
- Add multiple periods with start date and length
- View period history in interactive cards
- Predictive Insights: Next period date prediction
- Ovulation date and fertile window calculation
- Interactive Calendar


## Mini calendar highlights:
- Period days (pastel red)
- Fertile window (pastel green)
- Ovulation day (pastel orange)
- Touch-friendly design for mobile and iOS-like feel
